# Slide 4: Smart Search Function - 智能搜索功能

## Style Reference
Watercolor style - soft, hand-painted aesthetic with warm off-white background (#FAF8F0), visible brush strokes, and organic shapes.

## Content
Title: 智能搜索，精准匹配
Features:
- 输入症状：头晕、失眠等
- 输入食材：手边有什么
- 智能推荐：最匹配的食疗方案
- 视频指导：清晰的烹饪步骤

## Visual Composition
- Top: Hand-painted title
- Center: Large smartphone illustration with search interface
  - Search bar with "头晕" text
  - Suggested results with watercolor icons
- Left: Watercolor icons representing symptoms (dizzy icon, sleep icon)
- Right: Watercolor illustrations of ingredients (herbs, vegetables)
- Bottom: Play button icon with "视频步骤" text
- Background: Soft washes of dusty rose (#E8A0A0) and sky blue (#7EC8E3)
- Decorative: Connection lines with brush stroke style

## Color Scheme
- Background: Warm Off-White (#FAF8F0)
- Text: Warm Charcoal (#3D3D3D)
- Accents: Dusty Rose (#E8A0A0), Sky Blue (#7EC8E3), Sage Green (#87A96B)

## Notes
- Show intuitive, easy-to-use interface
- Use soft, friendly icon illustrations
- Demonstrate the smart matching process
