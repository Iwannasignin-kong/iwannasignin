# Slide 9: Core Features Summary - 核心功能总结

## Style Reference
Watercolor style - soft, hand-painted aesthetic with warm off-white background (#FAF8F0), visible brush strokes, and organic shapes.

## Content
Title: 五大核心功能
Features (icon + short text):
1. 智能搜索 - 症状/食材匹配
2. 个性化推荐 - 体质/节气定制
3. 视频指导 - 清晰易懂步骤
4. 社交分享 - 家庭健康关怀
5. 中医智慧 - 传统现代结合

## Visual Composition
- Top: Hand-painted title
- Center: Five feature icons arranged organically (not rigid grid)
  - Each icon painted with watercolor style
  - Magnifying glass (search)
  - Calendar/person silhouette (personalized)
  - Play button (video)
  - Share/heart icon (social)
  - Yin-yang or herbs (TCM wisdom)
- Connectors: Gentle curved lines with brush strokes
- Background: Soft washes of all accent colors
  - Sage green, soft coral, dusty rose, sky blue, soft lavender
- Decorative: Small nature accents (leaves, dots)
- Flow: Circular or organic cluster layout

## Color Scheme
- Background: Warm Off-White (#FAF8F0)
- Text: Warm Charcoal (#3D3D3D)
- Feature Accents: Sage Green (#87A96B), Soft Coral (#F4A261), Dusty Rose (#E8A0A0), Sky Blue (#7EC8E3), Soft Lavender (#C5B4E3)

## Notes
- Create visual variety with all accent colors
- Use organic, flowing layout
- Keep icons simple and hand-painted style
- Minimal text only
- Show interconnected features
